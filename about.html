Solar System Simulation is an interactive and educational web project designed to visually demonstrate how planets revolve around the Sun. Built using HTML, CSS, and JavaScript, this project offers a dynamic way to understand the orbital motion of celestial bodies in our solar system.

Inspired by old-school web games and fueled by a love for astronomy, the simulation blends fun with learning, making it ideal for students, educators, and curious minds alike. It also encourages feedback and community engagement through GitHub Pages.

The project was created by Abdul Moiez with a goal to spread accessible knowledge and spark scientific curiosity in young learners, educators, and space lovers around the world.
